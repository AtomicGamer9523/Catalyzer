Crate catalyzer

Expand description

Catalyzer is a web framework for Rust, made to be simple and easy to use.

§Example

use catalyzer::*;
 
#[main]
fn main() {
    App![index]
        .bind("0.0.0.0:3000")?
        .launch()
}
 
#[get("/")]
fn index() {
    "Hello, world!"
}
